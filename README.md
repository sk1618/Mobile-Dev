\# Mobile-Dev — StudySmart



StudySmart is a mobile study assistant application developed in Android Studio.



\## Features

\- Home dashboard

\- Planner

\- AI Coach

\- Statistics

\- Alerts and reminders



\## Backend (AI Coach)

The backend is built using Flask and provides roadmap generation for study topics.



\### Run backend

```bash

cd StudySmartBackend

python app.py

```



\### Backend endpoint

```text

http://10.0.2.2:5000/roadmap

```



\## Technologies

\- Java (Android)

\- XML layouts

\- SQLite

\- Flask (Python)
